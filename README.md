## Project Description

This repository contains my submission for the Expo/React Native calculator assignment. The project demonstrates basic Git workflows including initialisation, staging, committing, and pushing to a remote repository.

## Prerequisites

- Node.js (v18 or above)
- Expo CLI
- A device with Expo Go installed, or an Android/iOS emulator

## Installation

```bash
npm install
```

## Running the Development Server

```bash
npx expo start
```

Use the QR code printed in the terminal to open the app on your phone.
